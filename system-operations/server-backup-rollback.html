Server Backups & Rollback

Magento Open Source gives you the ability to back up different parts of the system—such as the database, file system, or both—and to rollback automatically. Files are compressed using the .gz and .tgz formats.

Best Practices

Restrict access to backup tools. Access to the backup and rollback tools can be restricted by configuring user permissions for backup and rollback resources. To restrict access, leave the corresponding checkbox unselected. If you need to grant access to the rollback resources, you must grant access to backup resources as well.

Backup before installing extensions and updates. Always perform a backup before installing an extension or updating Magento. Magento Connect Manager (MCM) includes a backup option that you can use before installing an extension or upgrading to a new release.
